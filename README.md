# 🪔 Diwali Sales Analysis

## 📌 Project Overview
This project analyzes Diwali sales data to uncover customer purchasing behavior and business insights.

## 🎯 Objectives
- Understand customer demographics
- Identify top-performing product categories
- Analyze spending patterns

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn

## 📊 Key Insights
- Married women aged 26–35 are the highest buyers
- Top states contribute majority of sales
- Food, Clothing, and Electronics are top categories

## 📁 Dataset
Diwali Sales Dataset (CSV)
